# ACFS5 — Kat op Saturnus

Een kat op de planeet Saturnus, met animerende LEDs aangestuurd door een ATtiny85.

| | |
|---|---|
| ![Artwork](fotos/acfs5-saturnus_artwork.png) | ![Lege PCB](fotos/acfs5-saturnus_pcb_leeg.jpg) |
| *Artwork* | *Lege PCB* |


## Beschrijving

De PCB toont een kat op de planeet Saturnus met zijn kenmerkende ringen. De LEDs verlichten zowel de ringen als de kat zelf.

## Stuklijst

| Aanduiding | Waarde / Type | Aantal |
|------------|--------------|--------|
| U1 | ATtiny85-20P | 1 |
| BT1, BT2 | AA of AAA batterijhouder | 2 |
| C1 | 100nF | 1 |
| D1–D8 | LED 3mm | 8 |
| D12–D14 | Bidirectionele LED 3mm | 3 |
| D9 | Bidirectionele LED 5mm | 1 |
| R1–R5 | 100Ω–680Ω* | 5 |
| SW1 | DIP-schakelaar 1-polig | 1 |
