# Benodigde gereedschappen en materialen

## Gereedschap

| Gereedschap | Opmerkingen |
|-------------|-------------|
| Soldeerbout | 25–40W met fijne punt (Ø 0,8–1mm) |
| Soldeertin | Ø 0,5–0,8mm, loodvrij of 60/40 lood-tin |
| Zijkniptang | Voor het afknippen van aansluitdraden |
| Pincet | Fijn puntig model voor SMD onderdelen |
| Multimeter | Voor het meten van weerstanden en controleren van verbindingen |
| PCB-houder | Derde hand of klem om de print vast te houden tijdens het solderen |

## Optioneel maar handig

| Gereedschap | Opmerkingen |
|-------------|-------------|
| Loep of vergrotingslens | Voor het controleren van SMD soldeerverbindingen |
| Desoldeerpomp | Voor het verwijderen van soldeertin bij fouten |
| Desoldeerwick | Alternatief voor desoldeerpomp |
| Flux (vloeimiddel) | Helpt bij moeilijke SMD pads |
| Isopropanol (IPA) | Voor het reinigen van de PCB na het solderen |

## ATtiny85 programmeren (Angry Cats From Space serie)

Voor het programmeren van de ATtiny85 heb je extra nodig:

| Onderdeel | Opmerkingen |
|-----------|-------------|
| Arduino Uno of Nano | Als ISP programmer |
| Jumperdraden | Voor de ISP verbinding |
| Arduino IDE | Met ATtiny85 board support (MicroCore of ATTinyCore) |

Zie de [Angry Cats From Space](../angry-cats-from-space/) pagina voor programmeernstructies.

## VFD Klok

De VFD klok werkt op **230V wisselspanning**. Zorg voor:

| Vereiste | Opmerkingen |
|----------|-------------|
| Kennis van 230V bedrading | Werk nooit aan de 230V zijde als de stekker er in zit |
| Passende behuizing | De 230V aansluitingen moeten goed beschermd zijn |
| Zekering | Aanwezig op de PCB (vervang alleen met de juiste waarde) |
